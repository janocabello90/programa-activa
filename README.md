# Programa ACTIVA — FisioReferentes

Landing page y assets de marca del Programa ACTIVA by FisioReferentes.

## Estructura

```
programa-activa/
├── index.html              # Landing page principal
├── assets/
│   ├── logos/              # Logos ACTIVA, FisioReferentes
│   ├── fotos/              # Fotos de Álex, equipo, eventos
│   └── icons/              # Iconos y recursos gráficos
└── README.md
```

## Logo

Una vez subido el logo a `assets/logos/logo-activa.png`, la landing lo carga automáticamente desde esa ruta.

## Despliegue

El archivo `index.html` es autocontenido (CSS + JS inline). Se puede servir desde cualquier hosting estático o integrar en la web de FisioReferentes.
